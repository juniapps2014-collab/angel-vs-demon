# Playtest Checklist 1-20

## Goal

`Stage 1~20`가 프로토타입 완료 기준을 만족하는지 확인한다.

## Core Flow

- `Login -> Lobby -> Battle` 진입이 끊기지 않는가
- 첫 전투 시작 방법이 직관적인가
- 클리어 후 `NEXT STAGE`와 `LOBBY` 흐름이 자연스러운가
- 패배 후 `RETRY` 흐름이 자연스러운가

## Stage Curve

- `Stage 1~3`: 조작과 자동 공격을 이해하기 쉬운가
- `Stage 4~6`: 폭탄 임프/헬하운드 등장 시 갑자기 막히지 않는가
- `Stage 7~9`: 적 조합이 늘어나도 강화 욕구가 생기는가
- `Stage 10`: 보스전 진입 전 웨이브 길이가 과하지 않은가
- `Stage 11~15`: 보스 클리어 후 다시 성장 구간으로 느껴지는가
- `Stage 16~19`: 엘리트 등장 이후 피로도가 과도하지 않은가
- `Stage 20`: 챕터 마감 보스전으로서 충분히 다르게 느껴지는가

## Growth Loop

- 2~3판 안에 `무기 강화`를 체감할 수 있는가
- 스킬 레벨업 비용이 초반 보상 대비 너무 무겁지 않은가
- 유물/스킬 장착 변화가 실제 전투 체감으로 이어지는가
- `골드 부족` 메시지가 반복 좌절보다 다음 목표로 작동하는가

## Combat Feel

- 검 자동 공격 사거리가 답답하지 않은가
- 피격, 처치, 스킬 발동이 시각적으로 구분되는가
- 보스 경고(`BOSS INCOMING`, `DANGER`, `PHASE SHIFT`)가 잘 보이는가
- 화면 흔들림이 과하지 않은가

## Persistence

- 재실행 후 닉네임이 유지되는가
- 재실행 후 `highestStage`, `gold`, `weaponLevel`이 유지되는가
- 재실행 후 `skillLevels`, `relicIds`, `skillIds`가 유지되는가
- Supabase 실패 시 로컬 폴백이 깨지지 않는가

## Bugs To Watch

- 존재하지 않는 적 타입으로 스폰되는가
- 웨이브가 끝났는데 보스가 나오지 않는 경우가 있는가
- 보스 처치 후 결과창이 뜨지 않는 경우가 있는가
- 데미지 숫자/이펙트가 너무 많이 쌓여 프레임이 떨어지는가

## Decision Rule

아래가 충족되면 `프로토타입 완료`로 본다.

- `Stage 1~20`를 명확한 진행 막힘 없이 플레이 가능
- `Stage 10`, `Stage 20` 보스전 체감 차별화 확인
- 성장/저장 루프가 한 사이클 정상 동작
- 치명적인 진행 버그 없음
