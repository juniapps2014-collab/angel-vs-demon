# 엔젤 vs 데몬 상세 콘텐츠 기획

## 1. 주인공 설계

### 기본 캐릭터

- 이름 가칭: 루미엘
- 소속: 천계 청소국 특수요원
- 역할: 근중거리 하이브리드 딜러
- 전투 방식: 이동은 수동, 공격은 자동 조준 및 자동 발사

### 전투 감각

- 기본 무기는 검
- 근거리에서는 연속 참격
- 일정 강화 이후에는 검기 파동 발사
- 적이 몰릴수록 광역 참격 효과가 강화되는 구조

## 2. 무기 설계

### 시작 무기: 라이트 세이버

- 타입: 검
- 기본 효과: 가장 가까운 적 자동 조준
- 강화 방향:
  - 공격속도 증가
  - 참격 범위 증가
  - 검기 발사 추가
  - 관통 수 증가

### 추후 무기 확장안

- 성검 트윈엣지: 이중 참격
- 코메트 블레이드: 검기 폭발
- 저지먼트 소드: 처치 수 기반 피해 증가

## 3. 스킬 설계

- `천벌 낙하`
  - 지정 범위에 광역 타격
- `홀리 대시`
  - 전방 돌진 후 경로 피해
- `스타 버스트`
  - 주변 다수 적 자동 타격
- `가디언 오라`
  - 일정 시간 피해 감소 및 체력 회복

## 4. 유물 설계

- `빛의 성배`
  - 연속 처치 수 비례 공격력 증가
- `부서진 악마 뿔`
  - 보스 대상 피해 증가
- `천계 나침반`
  - 자동 조준 범위 증가
- `웃음 가면`
  - 적 처치 시 낮은 확률로 폭발 발생
- `수호 깃털`
  - 피격 후 짧은 무적
- `황금 종`
  - 스테이지 보상 골드 증가

## 5. 적 설계

### 일반 적

- 웃는 좀비
  - 느리지만 숫자가 많음
- 폭탄 임프
  - 접근 후 자폭
- 데몬 슬라임
  - 독 장판 생성
- 헬하운드
  - 빠른 돌진
- 방패 좀비
  - 정면 피해 감소
- 투척 임프
  - 중거리에서 화염 구체 투척
- 독안개 슬라임
  - 이동 경로에 짧은 독 장판 생성
- 소환 사제
  - 약한 잡몹을 주기적으로 소환

### 엘리트 적

- 점프 광대 좀비
  - 넓은 범위 점프 공격
- 마계 드러머
  - 주변 적 공격속도 강화

### 적 역할 구조

- 물량 압박
  - 웃는 좀비
  - 슬라임
- 돌진 압박
  - 헬하운드
  - 엘리트 점퍼
- 공간 통제
  - 폭탄 임프
  - 독안개 슬라임
- 탱커/전열
  - 방패 좀비
- 원거리 견제
  - 투척 임프
- 버프/교란
  - 엘리트 드러머
  - 소환 사제

### 추가 적 1차 구현 우선순위

- `pitcher_imp`
  - 등장 추천: Stage 18+
  - 역할: 원거리 견제
  - 의도: 플레이어를 계속 움직이게 만들어 정지 플레이를 억제
- `toxic_slime`
  - 등장 추천: Stage 22+
  - 역할: 장판 압박
  - 의도: 좁은 지역에 오래 머무르지 못하게 만들어 이동 동선을 흔듦
- `summoner_priest`
  - 등장 추천: Stage 28+
  - 역할: 후방 교란 / 우선 처치 타깃
  - 의도: 잡몹 러시를 직접 늘리기보다 전투 우선순위를 만들기

### 실제 도입 타이밍

- `Stage 21`
  - `pitcher_imp` 첫 등장
- `Stage 25`
  - `summoner_priest` 첫 등장
- `Stage 30`
  - `toxic_slime` 첫 등장
- `Stage 50+`
  - 세 타입이 기존 적과 혼합되어 복합 압박 구간 형성

## 6. 로그인 및 프로필 UX 제안

### 로그인 방식

- 초반 MVP는 `익명 로그인 + 닉네임 생성`
- 이후 확장으로 이메일 로그인 또는 소셜 로그인 추가

### 프로필 필드

- 닉네임
- 아바타 아이콘
- 최고 스테이지
- 누적 처치 수
- 대표 무기
- 장착 스킬
- 장착 유물

### UX 흐름

1. 첫 실행
2. 익명 로그인 생성
3. 닉네임 입력
4. 로비 진입
5. 첫 스테이지 시작
