# 엔젤 vs 데몬 스테이지 1~20 밸런스 초안

## 설계 원칙

- 초반 20스테이지는 튜토리얼과 핵심 재미 검증 구간이다.
- 플레이어는 검 기반 자동 조준/자동 공격으로 적을 빠르게 학살해야 한다.
- 매 5스테이지마다 체감 난이도와 적 조합이 한 번씩 증가한다.
- 10, 20스테이지에 보스전 배치
- 11~20 구간은 적 수보다 `적 역할 혼합`으로 난이도를 올리고, 화면 과밀은 줄인다.

## 기준 수치

- 플레이어 시작 공격력: 30
- 플레이어 시작 체력: 300
- 플레이어 시작 공격속도: 초당 1.5회
- 기본 적 기준 처치 시간: 1~2초

## 보스 설계 초안

### Stage 10: 광대 좀비 대장

- 콘셉트: 풍선을 뿌리며 돌진하는 코믹 보스
- 패턴:
  - 전방 돌진
  - 미니 폭탄 소환
  - 체력 50% 이하 시 점프 충격파

### Stage 20: 헬 카니발 드러머

- 콘셉트: 드럼 소리로 적을 강화하는 쇼맨형 보스
- 패턴:
  - 박자 충격파
  - 일반 적 소환
  - 체력 40% 이하 시 공격속도 증가 오라

## 확인이 필요한 항목

- 초반 플레이어 체력이 너무 높거나 낮지 않은지
- 자동 조준 범위가 답답하지 않은지
- 보스 패턴이 캐주얼 유저에게 과하지 않은지
- 11~20에서 적 수는 줄었지만 긴장감이 유지되는지
