# Playtest Checklist 21-40

## Goal

`Stage 21~40`에서 신규 적 3종이 공정하고 읽기 쉬운 방식으로 작동하는지 확인한다.

## Enemy Readability

- `pitcher_imp`가 발사 전에 충분히 읽히는가
- `summoner_priest`가 우선 처치 대상으로 보이는가
- `toxic_slime` 장판이 깔리기 전에 예고가 보이는가
- 신규 적이 기존 적과 섞여도 무엇이 위험한지 구분되는가

## Stage Curve

- `Stage 21~24`: 원거리 압박이 답답함보다 긴장감으로 느껴지는가
- `Stage 25~29`: 소환 사제가 화면을 과도하게 채우지 않는가
- `Stage 30`: 신규 적 학습과 보스전이 과하게 겹치지 않는가
- `Stage 35`: 복합 조합이 피로감보다 전투 다양성으로 느껴지는가
- `Stage 40`: 보스전 압박이 혼란스럽지 않고 의도적으로 느껴지는가

## Fairness

- 탄에 맞았을 때 이유를 납득할 수 있는가
- 독 장판에서 HP가 깎일 때 피할 수 있는 정보가 있었는가
- 소환 사제가 숨기만 하고 잡기 어려운 위치에 반복 생성되는가

## Growth Loop

- `21~40` 구간에서 강화 보상이 난이도 증가를 따라가는가
- 신규 적 때문에 특정 스킬/유물만 강제되지 않는가
- 실패 후 다시 도전할 때 “다음엔 이렇게 하자”가 생기는가

## Performance / Clarity

- 투사체, 장판, 소환 이펙트가 너무 많이 겹쳐 프레임이 떨어지지 않는가
- HUD와 전장 효과가 겹쳐 중요한 정보가 가려지지 않는가
- 데미지 숫자와 소환 효과가 동시에 많아도 읽을 수 있는가

## Adjustment Priority

문제가 보이면 이 순서로 먼저 조정한다.

1. `pitcher_imp` 동시 등장 수
2. `summoner_priest` 소환 주기
3. `toxic_slime` 장판 지속시간
4. 신규 적 전용 이펙트 opacity
5. 스테이지 보상 수치
