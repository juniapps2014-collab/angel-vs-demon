# Supabase Notes

이 폴더는 Supabase 연동 관련 문서, SQL 스키마, 운영 메모를 저장하는 위치다.

초기 목표:

- 스테이지 데이터 외부화
- 적 및 보스 밸런스 수치 외부화
- 아이템, 스킬, 유물 정의 데이터 외부화

권장 다음 파일:

- `schema.sql`
- `seed.sql`
- `balance-notes.md`

초기 운영 원칙:

- 클라이언트 코드는 데이터 구조만 알고, 실제 수치는 서버 데이터로 읽는다.
- 밸런스 조정은 DB 값 변경으로 처리한다.
- 데이터 버전 필드를 추가해 클라이언트 캐시와 충돌을 줄인다.
