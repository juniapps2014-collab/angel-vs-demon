# Angel vs Demon

작업 폴더명은 `angel-vs-demon`, 게임 표기명은 `엔젤 vs 데몬`으로 사용한다.

## 현재 목표

- Cocos Creator + TypeScript 기반 아케이드 액션 게임 기획
- 웹 우선 개발 후 모바일 확장
- 스테이지 1~1000 운영형 구조 설계
- Supabase 기반 밸런스 데이터 업데이트 가능 구조 준비

## 현재 게임 콘셉트 요약

- 제목: 엔젤 vs 데몬
- 분위기: 캐주얼 코믹 좀비, 밝은 배경
- 전투: 검 기반 직접 공격, 자동 조준 및 자동 발사
- 성장: 무기, 스킬, 유물 조합
- 플레이 템포: 5~8분
- 전투 감각: 잡몹 학살 비중 큼
- 계정: 유저 로그인, 프로필, 진행도 저장 필요
